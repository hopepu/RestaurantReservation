## 💻 핵심 기능



#### 유저
- 로그인 및 회원가입
- 정보수정 및 회원탈퇴
- 식당 리스트 및 상세페이지 확인
- 리뷰 작성 및 수정

#### 관리자
- 기본정보로 가입
- 운영정보 등록, 메뉴등록 (가입과 별도로 이루어짐) 
- 정보 수정(가게정보, 메뉴)
- 회원 탈퇴 

#### 예약
- 레스토랑 정보로 휴일, 오픈마감시간 반영
- 유저 예약 확인
- 식당 예약 확인
- 예약 상태 확인(예약중, 예약확인, 예약완료)
